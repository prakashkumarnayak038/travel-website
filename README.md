# 🌍 YatraZone – Travel Website

A modern and responsive travel website built using HTML, CSS, and JavaScript.
This project allows users to explore destinations, view travel packages, and contact for bookings.

---

## 🚀 Features

* 🏠 Home page with 4K hero section
* 🌍 Destinations page with search functionality
* 💼 Travel packages with price & duration
* 📞 Contact form with validation & success message
* 🎯 Smooth scroll animations
* 🎨 Modern UI with gradient buttons & hover effects
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
